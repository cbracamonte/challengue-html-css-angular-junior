Javascript


Desarrolla los siguientes ejercicios en Javascript.

1. Crea una función saludar() la cual reciba como argumento la variable name y lo muestre como en un alerta, este debe ser invocado aquí, realiza las validaciones correspondientes como si es nulo y si es un string

2. Crea una función esPar() que reciba como parametro el arreglo [20,50,10,52,14,21,12,3,4,1,2,3,2] y me muestre en una alerta, los números pares, considerar validaciones.

3. Crea una función llamada hacerAlgo() que tome una función callback como argumento y llame a esa función después de un cierto período de tiempo (por ejemplo, 3 segundos). El resultado mostrarlo en consola y considerar validaciones si es necesario

4. Crea una función transformar() que reciba como parametro un array de 5 números, y estos deben ser multiplicados x5 en cada iteración, Se debe mostrar en consola el array original y el array transformado.

5. Usando fetch haz una petición al api de https://jsonplaceholder.typicode.com/todos/#id para mostrarme el resultado del post a petición del cliente, este debe activarse cuando de clic al botón Obtener Post y muestrame la información por consola

6. Utiliza setInterval para programar una función que se ejecute cada segundo e imprima un mensaje en la consola. Luego, utiliza clearInterval para detener la ejecución de la función después de 5 segundos.
